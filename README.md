# SketchUp Support Tools Interface

This project provides a custom interface inside SketchUp to simplify access to the main tools used by support, QA, developers, and 3D modelers.

The goal is to centralize essential features in a more accessible and efficient workflow, improving usability for teams that rely on SketchUp in their daily tasks.

## Technologies

### Backend
- Ruby (SketchUp plugin development)

### Frontend
- HTML
- CSS
- JavaScript
- React

## Purpose

This tool was created to make core SketchUp functions easier to access, helping different teams work faster and more efficiently within the software environment.

## Main Users
- Support team
- QA / Homologation team
- Developers
- 3D modelers

## Notes
This project integrates SketchUp plugin development with a modern frontend stack, combining Ruby on the backend with a React-based interface on the frontend.
